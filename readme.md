## 快捷键

快捷键         | 说明
---------------|----------------
Alt + d        | 垂直分屏 Pane
Alt + D        | 水平分屏
ALT + hjkl     | 切换 pane
ALT + HJKL     | pane 尺寸调整　
Alt + hj       | 切换 tab
Alt + HJ       | 移动 tab
Alt + 0~9      | 切换 tab
Alt + t        | 新建 tab
Alt + w        | 关闭 tab
Alt + x        | 关闭 tab
Alt + CTRL + f | 全屏
Alt + m        | 查看窗口列表
Alt + M        | 切换标签栏显示

## tmux 

`C-\`` 来切换到 tmux 配置, 快捷键相同

快捷键         | 说明
---------------|----------------
CMD + M        | 放大pane
